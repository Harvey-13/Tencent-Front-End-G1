## ✨ Features：

- babel
- react、react-dom
- typescript
- less
- husky
- eslint
- stylelint
- commitlint



## 🌸 Usage：

run :

```sh
// install modules
npm install

// development 
npm run dev

// build
npm run build
```



commit & add

```sh
// add
git add .

// commit 
npm run commit
```
